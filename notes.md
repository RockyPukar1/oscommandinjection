| Purpose of command    | Linux        | Windows   |
|------------------------|--------------|-----------|
| Name of current user   | whoami       | whoami    |
| Operating system       | uname -a      | ver       |
| Network configuration  | netstat -an  | netstat -an|
| Running processes      | ps -ef       | tasklist  |
